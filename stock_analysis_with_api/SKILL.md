# 股票分析与选股技能 - 结合实时API、历史数据与全方位分析

## 技能描述
这是一个综合性的股票分析与选股技能，能够结合实时API数据（如新浪财经）和历史数据（如tushare）进行全方位的量化分析和预测，包括艾略特波浪理论分析、K线形态分析、技术指标分析、基本面分析、市场情绪分析、舆情分析和智能选股。

## 功能特性
1. 实时股价获取
2. 历史数据获取
3. 艾略特波浪理论分析
   - 波浪结构识别（推动浪、调整浪）
   - 浪型划分与趋势结构判断（第1-5浪及A-B-C浪）
   - 推动浪特征识别（第1、2、3、4、5浪特征）
   - 调整浪特征识别（A、B、C浪特征）
   - 波浪趋势强度计算与分类
   - 斐波那契回撤与扩展位分析
   - 基于波浪理论的交易信号生成
4. K线形态分析
   - K线图解读（开盘、收盘、最高、最低价格）
   - 趋势判断（支撑与阻力位识别）
   - 常见形态识别（头肩顶、双底、三角形整理等）
5. 技术指标分析
   - 动量指标（RSI、MACD）
   - 波动率与成交量分析（布林带）
   - 移动平均线分析
6. 维克多·斯波朗迪策略分析（新增）
   - 123法则趋势反转识别
     * 趋势线突破检测
     * 不再创新高/新低确认
     * 关键高低点突破验证
   - 2B法则假突破识别
     * 假突破顶部检测（看跌2B）
     * 假突破底部检测（看涨2B）
     * 主力诱多/诱空行为识别
   - 综合反转信号置信度评估
7. 基本面分析
   - 财务报表分析（盈利能力、估值、现金流）
   - 行业分析与宏观经济影响
   - EPS、ROE、PE、PB等关键财务指标
8. 市场情绪与舆情分析
   - 成交量分析（量价关系、异常成交量识别）
   - 资金流向分析
   - 新闻舆情分析（情感分析、热度分析）
   - 社交媒体舆情分析（微博、雪球等平台）
   - 政策影响分析（对行业和个股的影响）
9. 智能选股工具
   - 价格区间筛选
   - 交易活跃度筛选
   - 技术指标筛选
   - 基本面筛选
   - 波浪理论筛选
   - 维克多·斯波朗迪策略筛选
   - 综合条件筛选
   - 股票排序与排名
10. 机器学习预测模型
11. 未来价格预测
12. 交易信号生成（结合波浪理论与维克多·斯波朗迪策略）
13. 投资建议（短期、中期、长期）
14. 综合舆情评分

## 技术栈
- Python
- tushare
- pandas
- numpy
- scikit-learn
- requests
- textblob
- jieba
- snownlp
- scipy (用于波浪理论分析)
- matplotlib (用于图表可视化)

## 数据源
- 实时数据: 新浪财经API
- 历史数据: tushare
- 财务数据: tushare财务报表接口
- 市场情绪: 基于历史成交量和价格数据
- 舆情数据: 新闻、社交媒体、政策公告等

## 预测模型
- 随机森林回归模型
- 支持多种技术指标作为特征
- R²评估模型准确性

## 输入要求
- tushare token
- 股票代码

- 当前股价信息

- 技术指标分析
- 基本面财务数据
- 市场情绪分析
- 舆情分析（新闻、社交媒体、政策影响）

- 投资策略（短期、中期、长期）

- 完整分析框架报告（按十大分析维度）
