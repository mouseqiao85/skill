# AI-Plat 项目

## 项目概述

AI-Plat 是一个集成了本体论驱动的认知连接、智能体系统和大模型驱动开发体验的下一代AI平台。本项目包含完整的开发文档、产品需求文档、平台实现代码以及相关工具。

## 目录结构

### 主要目录

- **docs/** - 项目相关文档
- **specs/** - 技术规格和设计规范
- **prd_docs/** - 产品需求文档 (PRD)
- **platform/** - 平台实现代码
- **utils/** - 实用工具脚本
- **archive/** - 历史归档文档
- **archives/** - 备份归档

### 文档组织方式

- **PRD文档**: 按版本分类保存在 `prd_docs/` 目录中
- **技术文档**: 保存在 `docs/` 目录中
- **代码实现**: 位于 `platform/` 目录中，按功能模块组织
- **工具脚本**: 放置在 `utils/` 目录中

## 版本信息

- **V1.0 (AI-Plat)**: 基础AI平台，包含本体论、智能体和Vibecoding三大核心能力
- **V2.0 (AI-Plat)**: 增加八层架构设计，引入MCP (Model Connection Protocol) 模块
- **V3.0 (NexusMind OS)**: 升级为AI操作系统，增强自主代理能力，引入智能决策层

## 重要文件

- `DIRECTORY_STRUCTURE.md` - 详细的目录结构说明
- 各版本的PRD文档 - 记录了产品演进过程
- 平台实现代码 - 包含完整的系统实现

## 使用说明

### 查找PRD文档
所有版本的PRD文档都集中在 `prd_docs/` 目录中，按版本命名清晰。

### 查找技术文档
各类技术文档保存在 `docs/` 目录中，包括开发指南、集成报告等。

### 访问源代码
平台的完整源代码位于 `platform/` 目录中，按功能模块组织。

## 维护指南

- 新增文档请放入对应的目录中
- 代码修改请在 `platform/` 目录下对应模块中进行
- 保持目录结构的整洁和一致性
- 定期归档旧文档到 `archive/` 目录
